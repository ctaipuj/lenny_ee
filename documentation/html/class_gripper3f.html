Detailed Description

This class controls the 3-finger robotiq gripper.

This class uses rostopics SModelRobotInput and SModelRobotOut to control the s-model robotiq gripper. This class subscribes to SmodelRobotInput in order to retrieve gripper's status information and publishes commands to the SModelOutput topic. Boost and robotiq_s_model_control are components required.
