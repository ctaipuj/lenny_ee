\subsection{Detailed Description}
This class controls the 3-\/finger robotiq gripper. 

This class uses rostopics S\+Model\+Robot\+Input and S\+Model\+Robot\+Out to control the s-\/model robotiq gripper. This class subscribes to Smodel\+Robot\+Input in order to retrieve gripper\textquotesingle{}s status information and publishes commands to the S\+Model\+Output topic.
